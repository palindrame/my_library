
DIMM  = Dual Inline Memory Module
SODIMM = Small Outline DIMM (smaller form factor)

DRAM is made up of transistors as well as capacitors.
The capacitors are used to hold charge, but charge can leak over time, hence Dynamic or DRAM.
DRAM requires 1 MOSFET (Metal Oxide Semiconductor Field Effect Transistor) and 1 capacitor for a bit.
SRAM on the other hand uses 6 transistors to hold a bit.

DRAM requires SENSE Amplifiers to handle the refresh Data that is lost over time.
Hence it eats up efficiency even if its small. Imagine we need to LOAD/STR but the DRAM is refreshing :).

DDR = Double Data Rate. Can handle data on both the rising edge as well as the falling edge

Physical RAM is Organized as:
Memory Channels -> DIMMS -> Ranks(which side) -> Banks(which square unit) -> Rows -> Cols
Row / Col is used since DRAM is layed out as a matrix.
