
                                                        +-------------------------------------------------------+
                                                        |          Physical Address (PA) for Memory Access      |
                                                        +-------------------------------------------------------+

                                                        After the CPU translates a Virtual Address (VA) through 
                                                        the page tables, you get a **Physical Address (PA)**.

                                                        🟢 This PA is what the **Memory Controller** actually uses to:
                                                          - Select a DRAM channel
                                                          - Pick the rank, bank, row, and column
                                                          - Fetch the correct byte(s)

                                                        It has nothing to do with the virtual address anymore.

                                                        The structure the memory controller cares about is not the same
                                                        as what the OS or MMU uses (like PFNs or page tables).
                                                        It depends on **DRAM architecture and configuration**.

                                                        The memory controller **DOES NOT CARE** about:
                                                          - Virtual addresses
                                                          - Page tables
                                                          - TLBs
                                                        It only sees the final **physical address** and splits it into:
                                                          Channel, Rank, Bank, Row, Column


                                                        ##########################################################

                                                                       PHYSICAL ADDRESS STRUCTURE
                                                  +----------------+-----------+---------+--------+-----------+-----------+
                                                  | Bits (PA)      | Channel   | Rank    | Bank   | Row       | Column    |
                                                  +----------------+-----------+---------+--------+-----------+-----------+
                                                  | [N:Z]          | Optional  | Varies  | Varies | Varies    | Varies    |
                                                  +----------------+-----------+---------+--------+-----------+-----------+

                                                        Note:
                                                        - High-order bits select **Channel/Rank/Bank**.
                                                        - Middle bits select **Row**.
                                                        - Lower bits select **Column** (or offset within row).

                                                        🧩 Example (simplified):
                                                          Physical Address (36 bits):
                                                            [35–34]: Channel (2 bits)
                                                            [33–32]: Rank (2 bits)
                                                            [31–28]: Bank (4 bits)
                                                            [27–14]: Row  (14 bits)
                                                            [13–0] : Column/Offset (14 bits)

                                                        The controller **decodes the PA** like this to drive the DRAM pins.

                                                              +-----------------------------------------------------+
                                                              |  Virtual Address → Translates to → Physical Address |
                                                              +------------------------+----------------------------+
                                                              | Page Table Entry (PTE) | Gives PFN (Page Frame Num) |
                                                              | VA[11:0]               | Page Offset                |
                                                              +------------------------+----------------------------+
                                                              | PA = (PFN << 12) | OR | VA[11:0]                    |
                                                              +-----------------------------------------------------+
